DevSecOps Course - Johns Hopkins University

Welcome to the DevSecOps Course!

This website is a Jekyll project created for the "Introduction to DevSecOps" course on Coursera, offered by Johns Hopkins University. The deployment of this project is managed by Huyen Tran.

DevSecOps is an essential practice that integrates security into the DevOps process, ensuring that security is an integral part of the software development and deployment pipeline.

This website will serve as a platform for sharing course materials, resources, and updates related to the course. Stay tuned for more information!
